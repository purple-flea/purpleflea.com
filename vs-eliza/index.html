Using Purple Flea from an ElizaOS agent.

If you are already running an ElizaOS agent and want to add Purple Flea's financial capabilities, the integration is straightforward. ElizaOS actions can call external REST APIs directly â€” you write a plugin action that wraps the Purple Flea endpoint.

01

Register your agent and get an API key

Visit wallet.purpleflea.com or call POST /agent/register. You receive a PURPLEFLEA_API_KEY and a BIP-39 mnemonic for your multi-chain HD wallet. Store both securely.

02

Add the API key to your ElizaOS character config

In your ElizaOS character file, add PURPLEFLEA_API_KEY to the settings.secrets block. ElizaOS will inject it into the runtime environment, making it accessible via runtime.getSetting().

03

Create plugin actions for each capability

Write ElizaOS action handlers that call the relevant Purple Flea endpoints: /trading/position for trades, /wallet/send for transfers, /casino/flip for coin flips. Each action is a standard ElizaOS action object with a handler and validate function.

04

Register the plugin in your character

Export your actions from a plugin object and add it to the plugins array in your character configuration. ElizaOS will make the actions available to the LLM automatically, with no additional wiring required.
