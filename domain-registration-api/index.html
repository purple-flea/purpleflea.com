API Reference

Six endpoints to manage
the full domain lifecycle.

All endpoints accept and return JSON. Authenticate with a Bearer token. Domain operations are idempotent where possible.

Method Endpoint Description
GET /domains/search Check availability and price. Query: ?domain=example.ai. Returns {available, price: {usdc, eth, sol}, tld_info, expires_at}
POST /domains/register Register a domain. Body: {domain, years: 1, payment_token: "usdc"}. Returns transaction hash and registration confirmation.
GET /domains/list List all domains owned by the authenticated agent. Returns [{domain, expires_at, auto_renew, nameservers, dns_records}]
PUT /domains/{domain}/dns Set or update DNS records. Body: {records: [{type, name, value, ttl}]}. Supports A, AAAA, CNAME, MX, TXT, NS, SRV.
POST /domains/{domain}/transfer Initiate an outbound transfer. Unlocks the domain and generates an EPP transfer authorization code.
POST /domains/renew Manually renew a domain. Body: {domain, years, payment_token}. Works for both manual and auto-renew override.

Propagation webhooks

DNS changes typically propagate globally within 5 minutes on Purple Flea's anycast network. Register a webhook URL and your agent receives a callback when propagation completes â€” no polling required.
