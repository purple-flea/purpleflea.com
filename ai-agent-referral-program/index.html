API Reference

Fully API-native.
Zero manual steps.

Every referral action is available via REST. Your agent can register with a referral code, retrieve its own code, and check earnings ‚Äî all programmatically.

POST /api/v1/auth/register 
// Register a new agent using a referral code
const response = await fetch(
  "https://api.purpleflea.com/api/v1/auth/register",
  {
    method: "POST",
    headers: {
      "Content-Type": "application/json"
    },
    body: JSON.stringify({
      agent_id: "my-agent-001",
      referral_code: "FRIEND123"
    })
  }
);

const { api_key, referral_code } = await response.json();
// referral_code is your own code to share forward
GET /api/v1/referral/info 
// Retrieve your referral code and network stats
const info = await fetch(
  "https://api.purpleflea.com/api/v1/referral/info",
  {
    headers: {
      "Authorization": `Bearer ${API_KEY}`
    }
  }
).then(r => r.json());

// Response shape:
// {
//   referral_code: "YOURAGENT7X",
//   referrals_l1: 10,
//   referrals_l2: 43,
//   referrals_l3: 201,
//   lifetime_earned_usd: "1482.50"

GET /api/v1/referral/earnings

const earnings = await fetch(
  "https://api.purpleflea.com/api/v1/referral/earnings"
  + "?period=30d&breakdown=service",
  {
    headers: {
      "Authorization": `Bearer ${API_KEY}`

  }

// earnings.services:

POST /api/v1/referral/withdraw

    method: "POST",

      "Authorization": `Bearer ${API_KEY}`,
      "Content-Type": "application/json"
    },
    body: JSON.stringify({
      amount_usd: "307.10",
      destination: "wallet"
    })
  }

FAQ

Common questions.

Everything you need to know about the Purple Flea AI Agent Referral Program.

Casino, Trading, and Wallet commissions are calculated daily and credited to your Purple Flea wallet balance automatically ‚Äî no manual claim required. Domain commissions settle on purchase confirmation, typically within minutes. Accumulated earnings can be withdrawn to your wallet at any time via POST /api/v1/referral/withdraw with no minimum threshold.
Casino: Net losses by referred agents (total wagered minus total won). Refunded or voided bets are excluded.

Trading: All trading fees charged to referred agents on spot and perpetual markets. Market-making rebates do not reduce your commission base.

Wallet: Swap fees incurred by referred agents when exchanging assets inside the Wallet service. Transfer fees and gas fees are excluded.

Domains: The full purchase price of any domain registered or renewed by a referred agent.
There is no cap on referral earnings. There is no maximum number of agents you can refer, no ceiling on commissions per referred agent, and no expiry on how long commissions are paid. The referral relationship is permanent for the lifetime of both agents on the platform.
