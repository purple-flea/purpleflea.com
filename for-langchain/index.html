from purpleflea_langchain import ( TradingGetPriceTool, TradingOpenPositionTool, WalletBalanceTool, ) from langchain.agents import create_openai_functions_agent from langchain_core.prompts import ChatPromptTemplate prompt = ChatPromptTemplate.from_messages([ ("system", """You are an autonomous trading agent. Check prices before opening positions. Never risk more than 2% of balance."""), ("human", "{input}"), ("placeholder", "{agent_scratchpad}"), ]) agent = create_openai_functions_agent( llm=ChatOpenAI(model="gpt-4o"), tools=[ WalletBalanceTool(), TradingGetPriceTool(), TradingOpenPositionTool(), ], prompt=prompt, )

Kelly Criterion Casino Agent

from purpleflea_langchain import CasinoFlipCoinTool from langchain.tools import tool @tool def kelly_bet(bankroll: float, edge: float) -> float: """Calculate optimal Kelly bet size.""" return bankroll * edge # Agent uses Kelly sizing before every flip tools = [kelly_bet, CasinoFlipCoinTool()] system = """You are a disciplined gambling agent. Always calculate Kelly bet before placing. Track your bankroll. Stop if down 50%.""" agent = initialize_agent( tools, llm, agent=AgentType.OPENAI_FUNCTIONS, system_message=system ) agent.run("Run 20 coin flips with Kelly sizing, starting bankroll 1 ETH")
How It Works

From zero to agent
in four steps.

01

Register your agent

Sign up at wallet.purpleflea.com or call POST /agent/register. You'll receive an API key and a BIP-39 mnemonic for your HD wallet.

02

Install the package

Run pip install purpleflea-langchain. Set PURPLEFLEA_API_KEY in your environment. No other dependencies beyond LangChain itself.

03

Pick your tools

Import only the tools your agent needs. Each tool is self-contained with a JSON schema, description, and error handling. Add them to your initialize_agent call.

04

Deploy and earn

Every API call your agent makes earns you a referral commission: 10% on wallet swaps, 20% on trading fees, 10% on casino, 15% on domains. Commissions accumulate in your wallet automatically.
