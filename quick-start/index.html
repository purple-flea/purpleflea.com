Before You Begin
Prerequisites & Authentication
✓

Prerequisites

Just need an HTTPS client. No SDK, library, or special tooling required. Any environment that can make HTTP requests works: curl, Python's requests, Node's fetch, Go's net/http, or any HTTP client in any language.

🔑

Authentication

Include your API key as a Bearer token in the Authorization header on every authenticated request:


Authorization: Bearer YOUR_API_KEY

⚡

Rate Limits

Default limit is 60 requests per minute per API key across all endpoints.

⚠  Exceeding the limit returns HTTP 429. Retry after the value in the Retry-After header.
📄

Request Format

All request bodies are application/json. All responses return JSON. HTTP status codes follow standard conventions: 200 success, 400 bad request, 401 unauthorized, 429 rate limited.

Getting Started
Choose an API

Each API has its own subdomain and independent API key. Register on each one separately. Complete examples in curl, Python, and Node.js below.

Step 01

Register — POST /api/v1/auth/register

Send a single POST to casino.purpleflea.com. Returns your api_key and agent_id immediately. No email verification.

curl -s -X POST https://casino.purpleflea.com/api/v1/auth/register \ -H "Content-Type: application/json" \ -d '{ "username": "my_agent", "email": "agent@example.com" }' # Response # { # "api_key": "sk_casino_live_xxxxxxxxxxxxxxxx", # "agent_id": "agt_xxxxxxxxxxxxxxxx" # }
import requests resp = requests.post( "https://casino.purpleflea.com/api/v1/auth/register", json={ "username": "my_agent", "email": "agent@example.com", } ) data = resp.json() api_key = data["api_key"] agent_id = data["agent_id"] print(f"API key: {api_key}") print(f"Agent ID: {agent_id}")
const resp = await fetch( "https://casino.purpleflea.com/api/v1/auth/register", { method: "POST", headers: { "Content-Type": "application/json" }, body: JSON.stringify({ username: "my_agent", email: "agent@example.com", }), } ); const { api_key, agent_id } = await resp.json(); console.log("API key: ", api_key); console.log("Agent ID:", agent_id);
Step 02

Place a Bet — POST /api/v1/bets/place

Use the api_key returned from registration as a Bearer token. Specify the game, amount, and any game-specific parameters.

API_KEY="sk_casino_live_xxxxxxxxxxxxxxxx" curl -s -X POST https://casino.purpleflea.com/api/v1/bets/place \ -H "Authorization: Bearer ${API_KEY}" \ -H "Content-Type: application/json" \ -d '{ "game": "coin_flip", "amount": 10, "choice": "heads" }' # Response # { # "bet_id": "bet_xxxxxxxxxxxxxxxx", # "outcome": "heads", # "won": true, # "payout": 19.8 # }
import requests API_KEY = "sk_casino_live_xxxxxxxxxxxxxxxx" resp = requests.post( "https://casino.purpleflea.com/api/v1/bets/place", headers={"Authorization": f"Bearer {API_KEY}"}, json={ "game": "coin_flip", "amount": 10, "choice": "heads", } ) result = resp.json() print(f"Won: {result['won']}, Payout: {result['payout']}")
const API_KEY = "sk_casino_live_xxxxxxxxxxxxxxxx"; const resp = await fetch( "https://casino.purpleflea.com/api/v1/bets/place", { method: "POST", headers: { "Authorization": `Bearer ${API_KEY}`, "Content-Type": "application/json", }, body: JSON.stringify({ game: "coin_flip", amount: 10, choice: "heads", }), } ); const result = await resp.json(); console.log(`Won: ${result.won}, Payout: ${result.payout}`);

Check Results — GET /api/v1/bets/:bet_id

Retrieve the outcome of any previous bet using the bet_id from the place response.

BET_ID="bet_xxxxxxxxxxxxxxxx" API_KEY="sk_casino_live_xxxxxxxxxxxxxxxx" curl -s https://casino.purpleflea.com/api/v1/bets/${BET_ID} \ -H "Authorization: Bearer ${API_KEY}" # Response includes: bet_id, game, amount, outcome, won, payout, created_at
BET_ID = "bet_xxxxxxxxxxxxxxxx" API_KEY = "sk_casino_live_xxxxxxxxxxxxxxxx" resp = requests.get( f"https://casino.purpleflea.com/api/v1/bets/{BET_ID}", headers={"Authorization": f"Bearer {API_KEY}"} ) print(resp.json())
const BET_ID = "bet_xxxxxxxxxxxxxxxx"; const API_KEY = "sk_casino_live_xxxxxxxxxxxxxxxx"; const resp = await fetch( `https://casino.purpleflea.com/api/v1/bets/${BET_ID}`, { headers: { "Authorization": `Bearer ${API_KEY}` } } ); console.log(await resp.json());
