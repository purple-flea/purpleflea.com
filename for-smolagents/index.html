What is smolagents

HuggingFace's lightweight
agent framework.

smolagents is HuggingFace's minimal, open-source agent framework built around a simple philosophy: agents should be small, readable, and composable. Unlike heavier frameworks, smolagents ships a clean Tool base class that any function or API can plug into. Purple Flea provides pre-built Tool subclasses for every Purple Flea endpoint â€” wallets, perpetual trading, provably fair casino, and domain registration â€” so your smolagents get real financial capabilities with zero boilerplate.

The purpleflea-smolagents package is designed to match the exact interface that smolagents expects: each tool declares a name, a natural-language description, typed inputs, and an output_type. Drop any combination into a CodeAgent or ToolCallingAgent and your agent can immediately check crypto balances, open leveraged positions, place provably fair bets, or register domains.

$ pip install purpleflea-smolagents
Tool Interface

Built on smolagents'
Tool base class.

smolagents defines agents through its Tool base class. Every Purple Flea tool is a proper subclass â€” meaning smolagents' CodeAgent can reason about it, call it, and chain results naturally.

tool_anatomy.py
from smolagents import Tool # How Purple Flea implements the smolagents Tool interface class WalletBalanceTool(Tool): name = "wallet_balance" description = """Get the current balance of the agent's crypto wallet. Returns balances across ETH, BTC, SOL, TRON, BNB, and MATIC. Use this before any transaction to check available funds.""" inputs = { "chain": { "type": "string", "description": "Blockchain to check: 'eth', 'btc', 'sol', 'tron', 'bnb', 'matic'", } } output_type = "string" def forward(self, chain: str) -> str: # Calls Purple Flea Wallet API and returns formatted balance string result = purpleflea_client.wallet.balance(chain=chain) return f"Balance on {chain}: {result['balance']} {result['symbol']}"
Code Example

CodeAgent with
Purple Flea tools.

smolagents' CodeAgent writes Python code to call tools â€” it reasons about which tools to use and chains results together. Purple Flea tools slot in naturally: the agent checks balance, calculates position size, opens the trade, all in one run.

from smolagents import CodeAgent, HfApiModel from purpleflea_smolagents import WalletBalanceTool, TradingOpenPositionTool # Configure your Purple Flea API key import os os.environ["PURPLEFLEA_API_KEY"] = "pf_sk_..." # CodeAgent writes Python code to call your tools agent = CodeAgent( tools=[WalletBalanceTool(), TradingOpenPositionTool()], model=HfApiModel("Qwen/Qwen2.5-72B-Instruct"), ) # The agent will: check ETH balance â†’ calculate safe position size â†’ open BTC long result = agent.run("Check my ETH balance and open a small BTC long position") print(result) # Agent output: "ETH balance is 1.24 ETH. Because smolagents' CodeAgent generates and executes Python code rather than making direct tool calls, it can perform multi-step reasoning: calling wallet_balance, computing 2% of the result, passing that computed value to trading_open_position, then summarising the outcome â€” all within a single agent.run() invocation. Multi-Agent Patterns

Orchestrator and
financial sub-agent.

smolagents supports a MultiStepAgent pattern where a high-level orchestrator delegates to specialised sub-agents. Purple Flea tools are a natural fit for a dedicated financial sub-agent.

multi_agent.py
from smolagents import CodeAgent, ToolCallingAgent, HfApiModel, ManagedAgent from purpleflea_smolagents import ( WalletBalanceTool, WalletSendTool, TradingOpenPositionTool, TradingGetPriceTool, TradingGetPositionsTool, CasinoFlipCoinTool, DomainsSearchTool, ) model = HfApiModel("Qwen/Qwen2.5-72B-Instruct") # Financial sub-agent handles all money operations finance_agent = ToolCallingAgent( tools=[ WalletBalanceTool(), WalletSendTool(), TradingOpenPositionTool(), TradingGetPriceTool(), TradingGetPositionsTool(), CasinoFlipCoinTool(), ], model=model, name="finance_agent", description="Handles crypto wallets, perpetual trading, and casino bets", ) # Orchestrator delegates financial tasks to the sub-agent orchestrator = CodeAgent( tools=[ManagedAgent(finance_agent), DomainsSearchTool()], model=model, ) result = orchestrator.run( """Check my portfolio, open a BTC long if ETH balance > 2 ETH, and search for purpleflea.io domain availability.""" )
Model Compatibility

Open-source or proprietary.
Your choice.

smolagents is model-agnostic by design. Purple Flea tools work with both HfApiModel for open-source models and LiteLLMModel for any commercial API.

open_source_model.py
from smolagents import CodeAgent, HfApiModel from purpleflea_smolagents import WalletBalanceTool # Use any open-source model from HuggingFace Hub agent = CodeAgent( tools=[WalletBalanceTool()], model=HfApiModel( "Qwen/Qwen2.5-72B-Instruct" # Also works with Llama-3.3-70B, Mistral, Deepseek ), )
commercial_model.py
