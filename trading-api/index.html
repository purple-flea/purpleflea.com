Leverage and Risk

Flexible leverage.
Transparent mechanics.

Every position is configured explicitly by your agent. There are no hidden defaults â€” leverage, margin mode, and order type are all set per request, so your agent has complete control over its risk profile at all times.

⚖

1x to 50x leverage

Set leverage from 1x (spot-equivalent exposure) up to 50x on major markets. Leverage is specified per order in the leverage field. Your agent can run different leverage levels on different positions simultaneously.

🛠

Isolated vs cross margin

Choose between isolated margin â€” where each position has a fixed collateral allocation and cannot affect other positions â€” or cross margin, where your full account balance backs all open positions. Isolated margin is recommended for autonomous agents to bound per-trade risk.

⚠

Liquidation mechanics

A position is liquidated when the mark price reaches the liquidation price â€” calculated from entry price, leverage, and maintenance margin rate. The API returns the liquidation price on every open position in GET /trading/positions. Your agent can monitor this in real time and reduce size before liquidation occurs.

🔄

Auto-deleveraging (ADL)

In extreme market conditions, Hyperliquid's ADL system may automatically close profitable positions on the opposite side to cover insolvent positions. The API surfaces the ADL queue rank for each of your agent's open positions so it can anticipate and manage this risk proactively.

Agent Trading Patterns

Strategies your agent
can run out of the box.

The Purple Flea Trading API is designed to support a range of autonomous strategies. Here are three common patterns with example logic your agent can implement directly.

Momentum

Trend-following strategy

Your agent polls GET /trading/price on a cadence, computes a short-term momentum signal (e.g. price vs 20-period EMA), and opens a long or short position when momentum exceeds a threshold. Positions are closed when momentum reverses or a profit target is hit.

Arbitrage

Funding rate arbitrage

When the 8-hour funding rate on a Hyperliquid market is strongly positive, your agent opens a short perpetual position and a corresponding long spot position via the Crypto Wallet API swap endpoint. The agent earns funding payments while remaining delta-neutral on price.

Hedge

Delta-neutral hedging

An agent holding a large spot position in ETH opens a proportional short ETH-PERP position to reduce directional exposure. As the spot position changes (e.g. due to DeFi activity), the agent recalculates and rebalances the hedge by adjusting position size via POST /trading/order.
