About MCP

What is the Model
Context Protocol?

The Model Context Protocol (MCP) is an open standard developed by Anthropic that lets AI models like Claude connect to external tools, data sources, and services through a standardised local interface. Instead of calling APIs through custom code, you configure lightweight MCP server processes that Claude can invoke natively ‚Äî both in Claude Desktop (the chat application) and programmatically through the Claude API. MCP servers communicate over stdio or HTTP, exposing a list of typed tool definitions that Claude can discover and call at runtime. This means Claude can check your crypto wallet balance, open a perpetual futures position, flip a provably fair coin, or register a domain simply by being asked in natural language ‚Äî with no custom integration code required on your side beyond a claude_desktop_config.json entry. Purple Flea ships four production-ready MCP servers ‚Äî one per product area ‚Äî so you can enable exactly the capabilities your workflow needs without installing anything you do not use.

Claude Desktop

Configure in sixty
seconds.

Open your claude_desktop_config.json file, add the Purple Flea MCP server entries below, paste in your API key, and restart Claude Desktop. All four servers are available immediately ‚Äî no build step, no Python environment, no dependencies to manage. The npx -y flag downloads and runs the latest server version automatically on each start.

config location (macOS): ~/Library/Application Support/Claude/claude_desktop_config.json
claude_desktop_config.json
{ "mcpServers": { "purpleflea-wallet": { "command": "npx", "args": ["-y", "purpleflea-mcp-wallet"], "env": { "PURPLEFLEA_API_KEY": "pf_sk_..." } }, "purpleflea-trading": { "command": "npx", "args": ["-y", "purpleflea-mcp-trading"], "env": { "PURPLEFLEA_API_KEY": "pf_sk_..." } }, "purpleflea-casino": { "command": "npx", "args": ["-y", "purpleflea-mcp-casino"], "env": { "PURPLEFLEA_API_KEY": "pf_sk_..." } }, "purpleflea-domains": { "command": "npx", "args": ["-y", "purpleflea-mcp-domains"], "env": { "PURPLEFLEA_API_KEY": "pf_sk_..." } } } }

Claude API

Use with the
Claude API too.

Purple Flea MCP servers work with the anthropic Python SDK's MCP integration. Define your servers in the API call and Claude can invoke Purple Flea tools within any programmatic conversation ‚Äî no Desktop app required.

claude_api_example.py
import anthropic import os client = anthropic.Anthropic(api_key=os.environ["ANTHROPIC_API_KEY"]) # Define which Purple Flea MCP servers Claude should have access to mcp_servers = [ { "type": "stdio", "name": "purpleflea-wallet", "command": "npx", "args": ["-y", "purpleflea-mcp-wallet"], "env": {"PURPLEFLEA_API_KEY": os.environ["PURPLEFLEA_API_KEY"]}, }, { "type": "stdio", "name": "purpleflea-trading", "command": "npx", "args": ["-y", "purpleflea-mcp-trading"], "env": {"PURPLEFLEA_API_KEY": os.environ["PURPLEFLEA_API_KEY"]}, }, ] # Claude can now call purpleflea_wallet_balance, purpleflea_trading_open_position, etc. response = client.beta.messages.create( model="claude-opus-4-6", max_tokens=1024, mcp_servers=mcp_servers, messages=[ { "role": "user", "content": """Check my ETH balance, then fetch the current BTC-PERP mark price. If my ETH balance is above 0.1, open a BTC-PERP long with 0.01 ETH at 5x leverage.""", } ], betas=["mcp-client-2025-04-04"], ) print(response.content)

How It Works

Set up in four steps.

01

Register and get your API key

Sign up at wallet.purpleflea.com. You will receive a pf_sk_... API key and a BIP-39 mnemonic for your agent's HD wallet. Keep the mnemonic safe ‚Äî it controls your funds across all six supported chains.

02

Add servers to your config

Open claude_desktop_config.json and paste in the MCP server entries shown above. Replace pf_sk_... with your actual key. Add only the servers you need ‚Äî each is independent and optional.

03

Restart Claude Desktop

Quit and reopen Claude Desktop. The MCP servers start automatically as background processes. You will see a tool icon appear in the Claude interface confirming that the Purple Flea tools are active and available.

04

Start using Purple Flea in chat

Ask Claude anything that requires the tools: "What is my ETH balance?", "Open a BTC long at 10x", "Flip a coin for 0.01 ETH", "Register cheapdomain.xyz if it is available". Claude handles tool selection and API calls automatically ‚Äî you just talk to it naturally.
