LangChain

The langchain-purpleflea package provides 20 LangChain BaseTool subclasses covering all four APIs. Pass them to any LangChain agent executor.

Install

pip install langchain-purpleflea

Quick start

from langchain_purpleflea import PurpleFleatoolkit from langchain.agents import AgentExecutor, create_openai_tools_agent from langchain_openai import ChatOpenAI toolkit = PurpleFleatoolkit( api_key="sk_live_...", referral_code="REF_xxx" # Embedded in sub-agent registrations ) # All 20 tools tools = toolkit.get_tools() # Or specific product tools casino_tools = toolkit.get_casino_tools() trading_tools = toolkit.get_trading_tools() wallet_tools = toolkit.get_wallet_tools() domains_tools = toolkit.get_domains_tools() llm = ChatOpenAI(model="gpt-4o") agent = create_openai_tools_agent(llm, tools, prompt) executor = AgentExecutor(agent=agent, tools=tools) result = executor.invoke({"input": "Open a long on ETH-PERP for $500 at 3x leverage"})

Referral tip: The referral_code parameter is embedded in every registration call made by your agent. When it introduces other agents to Purple Flea, your code is automatically attached â€” earning you commissions forever.

CrewAI

The crewai-purpleflea package provides @tool-decorated functions that work natively with CrewAI agents.

Install

pip install crewai-purpleflea

Quick start

from crewai import Agent, Task, Crew from crewai_purpleflea import ( trading_markets, trading_open_position, trading_close_position, casino_flip, casino_balance, casino_history, wallet_create, wallet_balance, wallet_swap, domains_search, domains_register ) trader = Agent( role="Quantitative Trader", goal="Open profitable leveraged positions based on technical signals", backstory="An AI agent with deep knowledge of perpetual futures markets", tools=[trading_markets, trading_open_position, trading_close_position] ) squatter = Agent( role="Domain Portfolio Manager", goal="Register valuable AI-related domains before they are taken", tools=[domains_search, domains_register] ) crew = Crew(agents=[trader, squatter], tasks=[...], verbose=True) result = crew.kickoff()

Direct REST API

Call any Purple Flea API directly from any language. JSON in, JSON out.

Python (requests)

import requests API_KEY = "sk_live_..." HEADERS = {"Authorization": f"Bearer {API_KEY}", "Content-Type": "application/json"} # Casino coin flip r = requests.post("https://api.purpleflea.com/api/v1/flip", json={"amount": 5.00, "side": "heads"}, headers=HEADERS) print(r.json()) # {"won": true, "payout": 9.95, "proof": "..."}

Node.js (fetch)

const res = await fetch("https://api.purpleflea.com/api/v1/flip", { method: "POST", headers: { "Authorization": `Bearer sk_live_...`, "Content-Type": "application/json" }, body: JSON.stringify({ amount: 5, side: "heads" }) }); const data = await res.json();

bash (curl)

curl -X POST https://api.purpleflea.com/api/v1/flip \ -H "Authorization: Bearer sk_live_..." \ -H "Content-Type: application/json" \ -d '{"amount":5,"side":"heads"}'
