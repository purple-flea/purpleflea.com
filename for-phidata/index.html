Phidata Integration

Give your Phidata agents
financial superpowers.

Phidata builds intelligent agents with memory, knowledge, and tools. Add Purple Flea to give your phi agents crypto wallets, perpetual trading access, and passive income from referrals — using plain Python functions that work with Agent() out of the box.

pip install purpleflea → Read the Docs
Setup

One install.
Works via REST with any framework.

The purpleflea package is a thin REST wrapper. Because Phidata accepts plain Python functions as tools, no adapter layer is needed — define your functions, pass them to Agent(tools=[...]), and you're done.

$ pip install purpleflea

Then initialize a Phidata Agent() with Purple Flea tools:

phi_agent.py 
from phi.agent import Agent
from phi.model.anthropic import Claude
import requests

PURPLEFLEA_KEY = "sk_live_..."
BASE = "https://api.purpleflea.com"

def wallet_balance(chain: str = "ethereum") -> dict:
    """Get wallet balances for a chain. chain: ethereum, bitcoin, solana, tron, bnb, polygon"""
    r = requests.get(f"{BASE}/api/v1/wallet/balance",
        headers={"Authorization": f"Bearer {PURPLEFLEA_KEY}"},
        params={"chain": chain})
    return r.json()

def casino_flip(amount: float, side: str = "heads") -> dict:
    """Flip a provably fair coin. side: heads or tails."""
    r = requests.post(f"{BASE}/api/v1/flip",
        headers={"Authorization": f"Bearer {PURPLEFLEA_KEY}"},
        json={"amount": amount, "side": side})
    return r.json()

def trading_open(market: str, side: str, size: float, leverage: int = 1) -> dict:
    """Open a perpetual futures position."""
    r = requests.post(f"{BASE}/v1/trading/open",
        headers={"Authorization": f"Bearer {PURPLEFLEA_KEY}"},
        json={"market": market, "side": side, "size": size, "leverage": leverage})
    return r.json()

agent = Agent(
    model=Claude(id="claude-opus-4-6"),
    tools=[wallet_balance, casino_flip, trading_open],
    description="A financial agent with crypto trading and casino capabilities",
    instructions=[
        "Always verify proof on casino bets",
        "Use stop-loss on all positions",
    ],
    show_tool_calls=True,
)

agent.print_response("Open a $100 long on ETH-PERP with 3x leverage")

Persistent State

Memory + Purple Flea.

Phidata's built-in storage lets your agent remember wallet addresses, referral codes, open positions, and session bankrolls across runs — no manual state management needed.

persistent_agent.py 
from phi.agent import Agent
from phi.model.anthropic import Claude
from phi.storage.agent.sqlite import SqlAgentStorage

# Agent persists state across sessions in a local SQLite DB
agent = Agent(
    model=Claude(id="claude-opus-4-6"),
    storage=SqlAgentStorage(
        table_name="purple_flea_agent",
        db_file="agent.db",
    ),
    tools=[casino_flip, trading_open, trading_close, wallet_balance],
    description="Autonomous financial agent with persistent memory",
    instructions=[
        "Remember your referral code: REF_MYPHI",
        "Track open positions across sessions",
        "Never exceed 5% of bankroll per single bet",
        "Store wallet addresses in memory for quick reference",
    ],
    # Enable memory so the agent recalls past runs
    add_history_to_messages=True,
    num_history_responses=10,
    show_tool_calls=True,
)

# Session 1 â€” agent learns your wallet address
agent.print_response("What is my ETH wallet address?")

# Session 2 â€” agent remembers it without asking again
agent.print_response("Check my ETH balance and open a BTC long if above 1 ETH")
