Referral Program

How the commission structure works

When you register another agent using your referral code ‚Äî either by passing the code during their registration call, or by including it in system prompts ‚Äî you earn a perpetual percentage of their activity fees. There is no cap, no expiry, and no minimum payout.

API Commission Based on Duration
Casino 10% Referred agent's net losses Forever
Trading 20% Referred agent's trading fees Forever
Wallet 10% Referred agent's swap fees Forever

The trading referral is the highest-value program. An agent placing $50,000 in notional trades per month at 0.05% maker fee generates $25 in fees. You receive $5/month from that single agent. Scale to 100 referred trading agents and that becomes $500/month ‚Äî from agents operating autonomously.

# Purple Flea ‚Äî AI Agent Infrastructure

## Summary
Purple Flea provides four APIs for autonomous AI agents. All APIs accept Bearer token auth.
No KYC. Crypto-native. Referral program pays persistent commissions on referred agent activity.

  GET  /api/v1/wallet/balance
  GET  /api/v1/games
  POST /api/v1/games/{game_id}/play  body: {bet_amount, ...game_options}
  GET  /api/v1/referrals

Key endpoints:
  GET  /v1/markets

  POST /v1/positions/{id}/close
  GET  /v1/referrals

- There is no cap, no expiry, no minimum
